# Smart Expense Analyzer & Spending Prediction

An end-to-end Machine Learning project that analyzes personal spending patterns and predicts future expenses using Python, Pandas, Scikit-learn, Matplotlib, and MySQL.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Expense Forecasting using Linear Regression and Random Forest
- Personalized Spending Recommendations
- MySQL Integration using SQLAlchemy
- Prediction Export to CSV
- Jupyter Notebook Implementation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- SQLAlchemy
- MySQL
- Jupyter Notebook

## Project Structure
- SmartExpenseAnalyzer.ipynb
- personal_expense_dataset.csv
- expense_prediction_model.pkl
- expense_predictions.csv
